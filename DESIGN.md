# DESIGN.md - 居家轻症引导问诊

## 气质与意象
**意象锚点**：清晨诊室，窗明几净，米白色纱帘透入柔和阳光。木质长桌上一杯温热的薄荷水，桌旁放着一本翻开的家庭健康手册。不是冷冰冰的传统医院，而是「云上家庭医生」式的温暖陪伴。

**关键词**：温和、清晰、有陪伴感、医疗但不冰冷。

## 视觉策略
- **图像方向**：极简插画风，少量线性 icon。强调"人形关怀"，而非"科技冷漠"。
- **图形语言**：圆角 12-16px 大块面 + 柔和投影（shadow-sm）+ 留白充足。
- **进度可视化**：三步法使用横向 stepper + 步骤气泡，进度一目了然。

## 配色方案
| 用途 | 色值 | 意象来源 |
| --- | --- | --- |
| 主色（医疗蓝） | `#3B82C4` | 清晨薄雾般的可靠蓝 |
| 主色-浅 | `#E6F0F8` | 纱帘透光的米白蓝 |
| 背景 | `#F7FAFC` | 米白纸张的底色 |

| 警示-就医 | `#DC2626` | 仅用于强就医提示，明确醒目 |
| 成功-可居家 | `#16A34A` | 安抚绿，确认居家可处理 |
| 警示-提示 | `#F59E0B` | 暖橙提醒，温和但要看到 |

## 字体排版
- **字体族**：PingFang SC / 思源黑体 / Microsoft YaHei（栈式降级）
- **节奏**：
  - H1：32-40px，semibold，呼吸感
  - H2：20-24px，medium
  - 正文：16px，line-height 1.7
  - 提示/免责：14px，颜色 muted

## 动效与交互
- 步骤切换：纵向 fade + 8px 位移，0.3s ease-out
- 选项 hover：边框颜色由灰转医疗蓝，过渡 0.2s
- 选中态：背景浅蓝渐变填充，描边 1.5px 主色
- CTA 按钮：实色医疗蓝 → hover 加深 8%
- 错误抖一抖：横向 4px 抖动 × 2 次，强提示"按步骤来"

## 页面结构
- **首页 = 问诊台**：上下结构
  - 顶部：Logo + 名称 + 简短说明
  - 中部：步骤气泡（三步横排）+ 标题 + 引导文案 + 选项区
  - 底部：导航按钮（上一步 / 下一步 / 查看方案）
- **结果页 = 居家方案卡**：6 块卡片堆叠，顶部条件允许时浮出"建议立即就医"红色横幅

## 组件规范
- `Stepper`：横向气泡，已完成=实心蓝，当前=呼吸光晕，未来=灰圈
- `OptionCard`：单选用大块卡片，多选用带勾选的卡片
- `RangeSlider`：痛感 0-10，配 emoji 表情辅助
- `AlertBanner`：三色变体（红=就医 / 橙=注意 / 绿=安心）
- `PlanSectionCard`：标题 + 序号 + 内容区，圆角 16px

## 设计禁忌
- ❌ 纯白底 + 纯黑字 + 硬朗线条（太冷）
- ❌ 大量红色、警示性颜色（制造焦虑）
- ❌ 花哨的科技感渐变（蓝紫发光、玻璃拟态）
- ❌ emoji 滥用（除痛感尺外，原则上不出现）
- ❌ 一屏塞满所有信息（违反"分步引导"原则）
